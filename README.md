# DoulRobotics KitchenMate RobotARM

## Versions

### 1. ESP32 Arduino Version
초기 RobStride MIT 제어 테스트 버전.

Path:
firmware/esp32_arduino/

### 2. STM32F407 Version
STM32F407 기반 CAN2 + USB CDC 제어 버전.

Path:
firmware/stm32_f407/

### 3. STM32 + SBC Version
STM32는 실시간 모터 제어, SBC는 상위 제어/UI/통신 담당.


## Motor IDs

- M1: RS03, ID 1
- M2: RS03, ID 2
- M3: RS02, ID 3
